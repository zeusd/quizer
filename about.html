Quizzer is a student project developed as part of a web technologies course. The goal of the project is to demonstrate dynamic content generation, data-driven interfaces, and basic statistical comparison between user results. Quizzes are defined using structured JSON files and rendered dynamically using JavaScript. This allows new quizzes to be added without modifying the core application logic. The project focuses on simplicity, transparency and modular design rather than user accounts or persistent personal data.
